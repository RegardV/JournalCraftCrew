# Journal Craft Crew - Repository Analysis Report

## Project Overview
Journal Craft Crew is an advanced, multi-agent system for automatically generating high-quality, personalized journaling guides. It uses multiple specialized AI agents working in concert to produce comprehensive 30-day journals and condensed 6-day lead magnets on any user-defined theme, with professional visual assets and output in PDF format.

## Application Architecture

### Multi-Agent System Components
1. **Manager Agent** - Orchestrates the entire content creation process
2. **Onboarding Agent** - Gathers user preferences and initializes project
3. **Discovery Agent** - Generates title options and analyzes implications
4. **Research Agent** - Performs evidence-based research using blog searches and NCBI PubMed
5. **Content Curator Agent** - Creates comprehensive journal content
6. **Editor Agent** - Refines content quality using sentiment analysis
7. **Media Agent** - Creates visual assets for journal content
8. **PDF Builder Agent** - Compiles final PDF documents

### Workflow Phases
- **Phase 1**: User Onboarding and Discovery
- **Phase 2**: Research and Content Creation
- **Phase 3**: Refinement and Production

### LLM Configuration
- Primary model: xai/grok-2-1212
- Base URL: https://api.x.ai/v1
- Temperature: 0
- Supports additional OpenAI and Zhipu AI models

## Installation and Setup
1. Clone the repository
2. Create and activate virtual environment
3. Install dependencies: `pip install -r requirements.txt`
4. Configure environment variables in .env file
5. Run application: `python main.py`
